Instructions for using API

  1. Use 'https://recommendation-system-api-aoml.herokuapp.com/recommend?name=movie_name' to get the response
  2. For space use '+' or '%20'
  3. 'https://recommendation-system-api-aoml.herokuapp.com/home' or 'https://recommendation-system-api-aoml.herokuapp.com/' to see the instructions
  4. The response from the API consist of movie name, Director, Genere, Actors of recommended movies followed by the same details of query movie
